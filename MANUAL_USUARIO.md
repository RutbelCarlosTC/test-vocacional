## 1. INTRODUCCIÓN
Este manual proporciona una guía detallada para el uso de la aplicación **CONOCET**, una herramienta diseñada para ayudar a los estudiantes a descubrir su perfil profesional mediante tests de orientación vocacional. La aplicación permite gestionar perfiles de usuario, realizar evaluaciones por áreas de conocimiento y visualizar resultados detallados mediante gráficos interactivos.

## 2. OBJETIVO
Orientar al usuario en el uso de las funcionalidades de la aplicación CONOCET para asegurar una experiencia de evaluación fluida, permitiéndole identificar sus fortalezas, intereses y personalidad en el ámbito académico y profesional.

## 3. ALCANCE
El presente manual cubre todas las funciones disponibles en la aplicación móvil/escritorio, incluyendo:
* Creación y selección de perfiles de usuario.
* Realización de evaluaciones por áreas.
* Consulta de resultados históricos y detallados.
* Interpretación de gráficos de personalidad y desempeño.

## 4. DESCRIPCIONES Y ABREVIATURAS
| Término / Sigla | Definición |
| :--- | :--- |
| **CONOCET** | Nombre de la aplicación de Test Vocacional. |

| **Podio de Resultados** | Visualización de las carreras o áreas con mayor puntaje obtenido. |

## 5. ACCESO AL SISTEMA
Al iniciar la aplicación, el usuario se encontrará con la pantalla de **Selección de Perfil**.
1. **Seleccionar Perfil:** Si ya tiene un perfil creado, toque el nombre o icono correspondiente.
2. **Crear Perfil:** Si es un usuario nuevo, haga clic en el botón de agregar para ingresar sus datos personales (Nombre, Edad, etc.).
3. **Inicio de Sesión:** Una vez seleccionado el perfil, el sistema cargará su historial y preferencias.

---

## 6. PANEL PRINCIPAL (INICIO)
Desde la pantalla de inicio, el usuario tiene acceso rápido a:
* Un resumen de su estado actual.
* El menú lateral (Drawer) para navegar por todas las secciones.
* Botones de acción para comenzar una nueva evaluación.

---

## 7. EVALUACIONES
Este módulo es el núcleo de la aplicación.
1. **Selección de Área:** El usuario debe elegir qué área desea evaluar (ej. Área de Ingenierías).
2. **Desarrollo del Test:** Se presentarán una serie de preguntas que deben responderse con sinceridad.
3. **Progreso:** La aplicación registra el avance de la evaluación actual.

---

## 8. RESULTADOS
Una de las secciones más completas, donde se visualiza el análisis del test:
* **Podio (Ranking):** Muestra las áreas o carreras donde el usuario obtuvo mayor afinidad.
* **Gráfico de Personalidad (Radar):** Visualiza rasgos como extroversión, analítica, creatividad, etc.
* **Gráfico de Macro Áreas (Pie Chart):** Distribución porcentual de los intereses vocacionales.
* **Tarjetas de Advertencia:** Avisos o sugerencias basadas en el análisis de las respuestas para mayor precisión.

---

## 9. MENÚ DE NAVEGACIÓN (APP DRAWER)
Ubicado en la parte superior izquierda, permite acceder a:
* **Inicio:** Regreso a la pantalla principal.
* **Evaluaciones:** Acceso directo a la selección de áreas.
* **Resultados:** Historial de resultados globales.
* **Acerca de:** Información sobre los desarrolladores y la versión de la app.
* **Cerrar Sesión:** Permite salir del perfil actual y regresar a la selección de usuarios.
